◆ getStatus()

SessionStatus org.vaultit.mobilesso.mobilessosdk.Session.getStatus ( )

Determines session status. If value returned is VALID, then tokens are in usable state, ie. refreshToken is valid and can be used to refresh the access token. If value returned is EXPIRED, then refresh token is expired, but browser's SSO session might still be valid, in which case a new authentication is not needed, ie. perform session check.

Returns
returns one of: VALID | EXPIRED | NO_SESSION

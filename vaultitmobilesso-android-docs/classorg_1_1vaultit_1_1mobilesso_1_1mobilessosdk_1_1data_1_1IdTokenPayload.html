Detailed Description

Convenience class created from ID Token. Contains user info and time stamps, etc. Function getValid() can be called to determine if JSON conversion succeeded.

Constructor & Destructor Documentation

org.vaultit.mobilesso.mobilessosdk.data.IdTokenPayload.IdTokenPayload ( @NonNull String  idTokenPayloadJSON,
@NonNull EnumSet< Scope >  scopes 
)

Constructor. Valid attribute (see isValid()) set to true if all mandatory values are found in id token payload string.

Mandatory values are: iat, exp, auth_time, iss, aud, sub, oxOpenIDConnectVersion, oxValidationURI

By specifying additional scopes, more fields will be attempted to be extracted from the idTokenPayload.

SCOPE FIELDS TO READ

VAULTIT_PROFILE vaultit_fullname, vaultit_firstname, vaultit_lastname, vaultit_gov_ids VAULTIT_ORG vaultit_org_name, vaultit_card_number, vaultit_gov_org_ids

isValid() will still return true if scoped fields are not read, ie. they are still considered optional.

Parameters
idTokenPayloadJSONid token payload string in JSON format
scopesthat determine what fields to read from idToken

MobileSSOAppDelegate

@objc public protocol MobileSSOAppDelegate: UIApplicationDelegate

Your AppDelegate must implement this protocol in order to make the login and logout flows work. Simply add the currentAuthFlow property to the AppDelegate with an initial value of nil.

  • currentAuthFlow

    The authorization flow used to present and handle the login process. Access is needed within the AppDelegate, which is why this needs to be in the UIApplicationDelegate of the host app.

    Declaration

    Swift

    var currentAuthFlow: OIDAuthorizationFlowSession?

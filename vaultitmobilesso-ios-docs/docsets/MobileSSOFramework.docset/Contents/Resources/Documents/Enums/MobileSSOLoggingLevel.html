MobileSSOLoggingLevel

@objc public enum MobileSSOLoggingLevel: Int

The logging level of the MobileSSO framework. A good value for application development is .warning and a good value for production is .none.

  • info

    All log messages are printed to the console.

    Declaration

    Swift

    case info
  • warning

    Warning level messages are printed to the console. This is recommended for application development, as the framework will warn you when doing things in a non-optimal way.

    Declaration

    Swift

    case warning
  • error

    Error level messages are printed to the console. These messages indicate either a direct misuse of APIs or fatal errors that occurred when using the API.

    Declaration

    Swift

    case error
  • none

    No messages will be printed from MobileSSOFramework.

    Declaration

    Swift

    case none
